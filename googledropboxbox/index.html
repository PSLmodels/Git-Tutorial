Git vs. Google Docs, Dropbox, and Box

As a last topic of background, we wanted to address a question that often often comes up in collaboration with other senior colleagues and students. “Why not just use Google Docs, Dropbox, or Box, Inc.?” These three file sharing and collaboration services could be loosely categorized under the version control system (VCS) rubric as centralized version control systems (CVCS). Refer to Chapter Version Control and Git History for more on version control methods.
